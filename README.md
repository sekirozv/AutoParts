# 🏎️ Информационная система «АвтоМагазин»

> **Дипломный проект** > Современное решение для автоматизации складского учета, управления продажами и анализа закупок в сфере автозапчастей.

---

## 🛠 Стек технологий

* **Язык:** C# (.NET 8)
* **Интерфейс:** WPF (Windows Presentation Foundation)
* **База данных:** MS SQL Server 2022
* **Архитектура:** Реляционная модель (12 связанных таблиц), паттерн кэширования представлений.

---

## 🚀 Быстрый старт

### 1. Подготовка базы данных
Приложение использует **MS SQL Server**. Для развертывания:
1. Откройте **SQL Server Management Studio (SSMS)**.
2. Нажмите правой кнопкой на «Databases» -> **Restore Database**.
3. Выберите файл устройства: `AutoParts.bak` и выполните восстановление.

### 2. Запуск приложения
Скомпилированный файл находится по пути:
`AutoParts\bin\Debug\AutoParts.exe`

## 🔐 Доступы (Тестовые данные)

В системе реализовано разграничение прав доступа (RBAC). Для входа используйте:

| Роль | Логин | Пароль | Возможности |
| :--- | :--- | :--- | :--- |
| **Администратор** | `admin` | `admin` | Полный доступ, управление складом и пользователями. |
| **Менеджер** | `manager` | `manager` | Оформление заказов, просмотр остатков, отчеты. |

---

## ✨ Ключевые особенности

* **Безопасность:** Хэширование паролей по алгоритму **SHA-256**.
* **Целостность данных:** Система «мягкого удаления» (архив) и строгие связи SQL Foreign Keys.
* **Производительность:** Мгновенная навигация за счет кэширования UI-компонентов.
* **Отчетность:** Динамическая генерация печатных форм через **FlowDocument**.
* **UX/UI:** Адаптивный интерфейс, разработанный на основе прототипов в Figma.
